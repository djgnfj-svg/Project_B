# Project_B

2D **웹 게임** (Godot 4.7.1, Web/HTML5 익스포트 타깃). 현재 **초기 단계** — 기획과 하네스 뼈대만 잡혀 있습니다.

## 시작하기

1. **Godot 4.7.1**을 직접 받으세요 — 엔진 바이너리는 용량이 커서 저장소에 넣지 않습니다(`.gitignore` 처리).
   - https://godotengine.org/download → 4.7.1 (또는 4.7.x)
2. 저장소를 클론한 뒤 Godot으로 이 폴더를 엽니다. (아직 `project.godot`가 없으니 곧 추가됩니다.)

## 문서

- **`docs/GDD.md`** — 게임 기획서(단일 진실원). 게임이 무엇인지는 여기서 정합니다. 🔒 수정은 승인제.
- **`CLAUDE.md`** — 개발 규율·아키텍처·에이전트 하네스 안내(구현 정본).

## 개발 메모

- 언어: GDScript (typed).
- 웹 게임이라 **네트워크가 코어** — 네트워크 코드는 항상 리뷰를 거칩니다(WebSocket·서버 권한·입력 검증).
- `.mcp.json`의 `GODOT_PATH`는 로컬 절대 경로라, 각자 환경에 맞게 바꿔야 합니다.
