# Project_B — 게임 기획서 (GDD)

> 🔒 **이 문서는 Project_B의 유일한 기획 진실원(single source of truth)이다.**
> - **게임이 "무엇인가"**(비전·핵심 루프·메커니즘·콘텐츠·시스템)는 전부 여기서 결정된다. 다른 문서·코드·대화와 충돌하면 **이 문서가 이긴다.**
> - **수정에는 사용자 허락이 필요하다.** `projectb-planner`(및 누구든)는 이 문서를 고치기 전에 **변경안(diff)을 먼저 제시하고 승인을 받는다.** `.claude/settings.json`의 권한 규칙으로도 Edit/Write가 매번 확인을 요구하도록 잠겨 있다.
> - 엔지니어링/아키텍처 정본은 `CLAUDE.md`다. 이 문서는 **기획**, CLAUDE.md는 **구현 규율** — 역할이 다르다.
