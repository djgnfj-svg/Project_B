B안 — 숲속 개간지 · 모닥불 광장 (세피리아풍 유기적 배치)

의도: 마을 전체가 숲에 둘러싸인 개간지. 길이 직선이 아니라 굽이치는 흙길로 스폰(좌하) → 모닥불 광장(중앙) → 게이트(우측)를 잇는다. 건물은 격자가 아니라 비스듬히 흩어놓아 "자연스럽게 자란 마을" 느낌. 랜턴·창문 불빛의 따뜻한 글로우, 연못·꽃밭·그루터기 장식.

C안 — 개울 마을 · 나무다리 (세피리아풍 + 벚꽃 포인트)

의도: 개울이 마을 남쪽을 가로지르고 나무다리로 남쪽 꽃밭 쉼터와 연결. 북쪽에 기능 건물 3채, 곡선 길이 처마 밑을 지나며 이어진다. 세피리아 특유의 벚꽃나무 포인트 컬러(연분홍)를 2그루 배치해 화면에 온기를 더함. 물가 갈대·수련, 랜턴 글로우.
