## A. 구조 (최우선 — 효과 최대)

- [ ] **A1. Y-소트 도입** — 마을·스테이지 루트에 `y_sort_enabled`, 소품/캐릭터의 정렬 기준점을 발밑으로 통일(현재 소품이 좌상단 앵커라 offset 정리 필요). 플레이어가 나무 뒤로 돌면 가려지는 것만으로 씬이 입체가 된다. *(근거: 2d-essentials §1 그리기 순서)* — **효과 ★★★ / 비용 중**
- [ ] **A2. 마을 바닥 960×576 리베이크** — 기존 타일 에셋(grass/path/bankrock/water)으로 확장판을 다시 굽는다(projectb-art). 리베이크 후 `Ground scale=1.5` 제거 + `village.gd MAP_RECT` 주석 미러 갱신. — **효과 ★★★ / 비용 중 (아트 위임)**
- [ ] **A3. 스테이지 배경 밀도** — 바닥 타일 변형 2~3종 + 데칼(돌·풀포기·꽃·낙엽) 흩뿌리기 + 가장자리(절벽/수풀 경계) 처리. 마을에서 이미 증명된 방법 그대로. — **효과 ★★★ / 비용 중**

## B. 조명·후처리 (분위기)

- [ ] **B1. 풀스크린 후처리 — 비네트 + 은은한 색보정** — CanvasLayer + ColorRect + canvas_item 셰이더 1장(전 씬 공용). 화면 가장자리를 살짝 어둡게만 해도 "게임 화면"이 된다. *(shader-basics §6 후처리 표준 패턴)* — **효과 ★★☆ / 비용 소**
- [ ] **B2. 씬별 시간대 톤 확립** — 마을=따뜻한 낮(적용됨), 스테이지1·2=서늘한 숲, 보스=붉은 어스름. CanvasModulate 색만 씬별로 다르게 — 챕터 진행의 감정 곡선이 색으로 생긴다. — **효과 ★★☆ / 비용 극소**
- [ ] **B3. PointLight2D 실광원 실험** — 화덕·모닥불에 진짜 2D 라이트 + 주변만 밝게(밤 톤과 조합 시 극적). ⚠ Compatibility 렌더러+웹에서 성능 실측 필수, 안 되면 현 가산 글로우 유지. *(2d-essentials §4)* — **효과 ★★☆ / 비용 중 (검증 필요)**

## C. 모션·파티클 (살아 있음)

- [ ] **C1. 모닥불 진짜 불꽃** — 현 정지 스프라이트+글로우 → 불 파티클(뜨거운 색 그라디언트 + 축소 커브, `preprocess` 필수). 마을 모닥불·챕터 모닥불 공용 씬으로. *(particles-vfx §4 불 레시피)* — **효과 ★★☆ / 비용 소**
- [ ] **C2. 강물 UV 스크롤** — 강 표면에 흐르는 물결 셰이더(UV 스크롤 + 미세 왜곡). 현 반짝 파티클과 조합. *(shader-basics §3 UV 스크롤)* — **효과 ★★☆ / 비용 소**
- [ ] **C3. 바람 — 나무·수풀 흔들림** — canvas_item vertex 셰이더로 상단만 좌우 sway(TIME 기반, 나무마다 위상 오프셋). 정지화면이 사라진다. — **효과 ★★☆ / 비용 소**
- [ ] **C4. 앰비언트 입자** — 마을: 반딧불(저녁 톤 시)·민들레 홀씨. 스테이지: 떠다니는 먼지·낙엽. `fixed_fps=30`, `visibility_rect` 설정(웹 성능). *(particles-vfx §10)* — **효과 ★☆☆ / 비용 소**
- [ ] **C5. 이동 먼지 퍼프 확대** — feel층의 플레이어 먼지를 잔몹 이동·NPC에도. — **효과 ★☆☆ / 비용 극소**

## D. 애니메이션 커버리지

- [ ] **D1. 코볼트·드레이크 애니** — 고블린 파이프라인(12프레임 스트립+SpriteFrames) 재적용. 화면에 정지 몬스터가 남지 않게. — **효과 ★★☆ / 비용 중 (아트 위임)**
- [ ] **D2. NPC idle 숨쉬기 2프레임** — 6종 × 2프레임. 마을이 정지화면에서 벗어남. — **효과 ★★☆ / 비용 소~중**
- [ ] **D3. 보스(브루트) 애니** — 보스전이 핵심 콘텐츠(GDD §2)인데 정지 스프라이트면 최종 인상이 깎인다. — **효과 ★★★ / 비용 중**

## E. UI 테마

- [ ] **E1. 공용 Theme 리소스** — 기본 회색 Godot 테마 탈피: 패널 9-slice(나무/양피지 톤), 버튼 상태색, Galmuri9 일괄 적용. 제작/인벤/로비가 한 벌로 통일. *(godot-ui 테마 시스템)* — **효과 ★★☆ / 비용 중**
- [ ] **E2. 힌트 라벨 칩** — 게이트/제작대 "F — " 라벨에 반투명 배경 칩 + 외곽선(가독성). — **효과 ★☆☆ / 비용 극소**

## F. 오디오 앰비언스

- [ ] **F1. 마을 앰비언스 루프** — 새소리·강물 소리(gen_sfx 절차 생성 또는 CC0), Audio 오토로드 버스에. 화면 밖 정보가 생기며 공간감이 붙는다. — **효과 ★★☆ / 비용 소**
- [ ] **F2. BGM** — 마을/스테이지/보스 3트랙 (외부 CC0 소스 검토). ⚠ 웹 autoplay — 첫 입력 후 시작 규약 유지. — **효과 ★★★ / 비용 외부 의존**
